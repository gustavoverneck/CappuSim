LatteLBM simulates fluid dynamics using the Lattice Boltzmann Method (LBM), a mesoscopic approach based on particle distributions.

üîÅ LBM Update Cycle

Each time step performs the following operations:

1. Collision ‚Äì relaxes the distribution toward equilibrium:

2. Streaming ‚Äì shifts each distribution f[q] to the neighboring cell along its direction c[q].

3. Swap ‚Äì replaces the old distribution with the streamed one (f_new ‚Üí f) for the next iteration.
